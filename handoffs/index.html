    Handoffs

    Handoffs allow agents to delegate tasks to other agents when they need specialized capabilities or when a task is better handled by a different agent.

    Basic Handoff

    import { Agent, Runner } from 'openai-agents';

// Create specialized agents
const mathAgent = new Agent({
  name: "MathExpert",
  instructions: "You are a math expert that can solve complex mathematical problems."
});

const codingAgent = new Agent({
  name: "CodeExpert",
  instructions: "You are a programming expert that can help with coding tasks."
});

// Create main agent with handoff capabilities
const mainAgent = new Agent({
  name: "Coordinator",
  instructions: "You are a coordinator that can delegate tasks to specialized agents.",
  handoffs: [mathAgent, codingAgent]
});

// Run the main agent
const result = await Runner.run(mainAgent, "Solve the equation: 2x + 5 = 15");

    Handoff Configuration

    Required Parameters

    • name: A unique identifier for the agent
    • instructions: The system prompt that defines when to use this agent
    • handoffs: Array of agents that can be delegated to

    Optional Parameters

    • handoffRules: Rules for when to use each agent
    • handoffPrompt: Custom prompt for the handoff process
    • handoffTimeout: Maximum time to wait for handoff completion

    Handoff Types

    1. Direct Handoff

    Simple delegation to another agent:

    const result = await Runner.run(mainAgent, "Task", {
  handoff: mathAgent
});

    2. Conditional Handoff

    Handoff based on conditions:

    const mainAgent = new Agent({
  name: "Coordinator",
  instructions: "Delegate tasks based on their nature",
  handoffs: [mathAgent, codingAgent],
  handoffRules: {
    mathAgent: (input) => input.includes("solve") || input.includes("calculate"),
    codingAgent: (input) => input.includes("code") || input.includes("program")
  }
});

    3. Chain Handoff

    Multiple agents working in sequence:

    const result = await Runner.run(mainAgent, "Task", {
  handoffChain: [mathAgent, codingAgent]
});

    Best Practices

    1. Clear Instructions: Each agent should have clear instructions about its role
    2. Proper Delegation: Only delegate when necessary
    3. Error Handling: Handle handoff failures gracefully
    4. Timeout Management: Set appropriate timeouts for handoffs
    5. Monitoring: Track handoff performance and success rates

    Advanced Features

    Handoff Filters

    Filter when handoffs should occur:

    const mainAgent = new Agent({
  name: "Coordinator",
  instructions: "Delegate tasks based on their nature",
  handoffs: [mathAgent, codingAgent],
  handoffFilters: {
    mathAgent: (input) => {
      return input.includes("solve") || input.includes("calculate");
    },
    codingAgent: (input) => {
      return input.includes("code") || input.includes("program");
    }
  }
});

    Handoff Context

    Pass context between agents:

    const result = await Runner.run(mainAgent, "Task", {
  handoff: mathAgent,
  context: {
    previousSteps: ["Step 1", "Step 2"],
    userPreferences: { precision: "high" }
  }
});

    Handoff Monitoring

    Monitor handoff performance:

    const result = await Runner.run(mainAgent, "Task", {
  handoff: mathAgent,
  onHandoffStart: (agent) => {
    console.log(`Handoff started to ${agent.name}`);
  },
  onHandoffComplete: (agent, result) => {
    console.log(`Handoff completed to ${agent.name}`);
  }
});
